## satoWa

ペットの日常の写真を投稿することで、里親の定期報告を簡単に管理できるサービスです。

- [予定機能](https://magic-ozraraptor-aa8.notion.site/f976bca28c024b1bacf2d5eaffcf1608)
- [定期報告に関するアンケート](https://forms.gle/EirFfRS9ryNfqdvL7)
